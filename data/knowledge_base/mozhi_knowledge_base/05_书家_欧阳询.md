## 二、生平经历

欧阳询出身于南朝官僚家庭，其父欧阳纥为南朝陈朝将领。幼年因父谋反被杀，家族蒙难，由江总收养。虽身世坎坷，但勤学不辍，博览经史，尤精书法。

入唐后，欧阳询官至太子率更令、弘文馆学士，人称"欧阳率更"。他长相貌寝（相貌不佳），但才华出众，与裴矩、陈叔达等编修《艺文类聚》。

### 传奇故事
传说欧阳询路见晋代书法家索靖所书石碑，初见觉得平常，走了一段又返回再看，越看越觉精妙，最后竟在碑旁铺毡坐卧，研习三日三夜方才离去。这种对书法的痴迷，正是他成为一代宗师的原因。

## 三、书法成就

### 总体风格
欧阳询的楷书以**法度森严、结构险峻、笔力刚劲**著称。其书学来源广泛，融合北碑的雄强与南帖的秀逸，自成一家。

### 艺术特点
1. **结构精妙**：中宫收紧，四肢伸展，"险中求稳"，在平正中求变化
2. **用笔刚健**：起收笔干净利落，棱角分明，骨力洞达
3. **法度严谨**：每字每笔皆有法可循，是学习楷书的最佳范本
4. **兼善诸体**：楷、行、草、隶、篆无所不能

## 四、代表作品

### 楷书三大名碑

#### 1. 《九成宫醴泉铭》（最著名）
| 项目 | 内容 |
|------|------|
| **年代** | 唐贞观六年（632年），欧阳询75岁时书 |
| **背景** | 唐太宗在九成宫避暑，发现醴泉，魏征撰文，欧阳询书丹 |
| **特点** | 法度最为严谨，结构最为精到，被誉"楷法极则" |
| **地位** | 欧阳询楷书最高成就，千年来学习楷书的经典范本 |
| **现存** | 陕西麟游县，有残损 |

#### 2. 《化度寺碑》
| 项目 | 内容 |
|------|------|
| **年代** | 唐贞观五年（631年） |
| **特点** | 笔力劲健，体势方峻，较《九成宫》更为内敛含蓄 |
| **地位** | 欧阳询早期楷书代表作，清代翁方纲推为欧书第一 |

#### 3. 《皇甫诞碑》
| 项目 | 内容 |
|------|------|
| **年代** | 唐贞观年间 |
| **特点** | 结构更为险峻，用笔更为锐利 |
| **地位** | 欧阳询楷书中最见骨力之作 |

### 行书作品
- **《仲尼梦奠帖》**：行书代表作，笔力苍劲，意态从容
- **《千字文》**：行书范本，流传广泛

## 五、欧阳询的书论贡献

欧阳询不仅是大书家，也是书法理论家。相传他著有：

- **《八诀》**：论述楷书基本笔画的写法要领
- **《传授诀》**：讲述书法的学习方法和心得
- **《用笔论》**：探讨用笔技巧
- **《三十六法》**：论述楷书结字法则（一说后人伪托）

这些书论对后世书法教育产生了深远影响。

## 六、欧体学习指南

### 适合人群
- 书法初学者（楷书入门首选之一）
- 希望打下严谨法度基础的学习者
- 喜欢结构精妙、骨力清刚风格的人

### 推荐学习顺序
```
《九成宫醴泉铭》→ 《化度寺碑》 → 《皇甫诞碑》
（入门精研）     （进阶深入）   （挑战骨力）
```

### 学习要点
1. **注重结构**：欧体的精髓在于结构，要仔细观察每个字的空间分布
2. **把握中宫**：体会"中宫收紧、四肢伸展"的结字规律
3. **锤炼笔力**：欧体用笔刚健，要注意起收笔的果断
4. **避免刻板**：法度严谨不等于死板，要注意内部的微妙变化

## 七、历史评价

> "欧阳询八体尽能，笔力劲险，篆体尤精。飞白冠绝，峻于古人。" —— 张怀瓘《书断》

> "询书初学王羲之，后渐变其体，笔力险劲，为一时之绝。" —— 《旧唐书》

欧阳询的楷书对后世影响极为深远，其书体被称为"欧体"，与颜真卿的"颜体"、柳公权的"柳体"、赵孟頫的"赵体"并称"楷书四大家"。初唐以降，学楷书者几乎无不从欧体入手或受其影响。
