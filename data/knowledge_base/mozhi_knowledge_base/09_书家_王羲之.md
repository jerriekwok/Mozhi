## 二、生平经历

王羲之出身魏晋名门琅邪王氏。家族中书法人才辈出，伯父王导、王敦，叔父王廙皆善书法。

### 重要经历
- 曾任秘书郎、征西将军庾亮参军、长史，后拜右军将军、会稽内史，世称"王右军"
- 永和十一年（355年）称病辞官，隐居会稽山阴（今浙江绍兴），专心于书法
- 与名士谢安、孙绰、支遁等交游，雅好山水

### 东床快婿
太尉郗鉴派人到王家选婿，王家子弟皆正装以待。唯有王羲之袒腹卧于东床，若无其事。郗鉴闻之，说："这正是我要选的好女婿！"于是将女儿嫁给他。这就是"东床快婿"典故的由来。

### 入木三分
相传王羲之在木板上写字，木工刻字时发现墨迹透入木板有三分深。"入木三分"形容其笔力之强劲。

### 竹扇题字
王羲之见一老妇卖六角竹扇，便在她扇上各题五字。老妇起初不悦，王羲之说："但言是王右军书，以求百钱。"果然众人争相购买。后老妇又找他题字，王羲之笑而不答。

## 三、书法成就

### 总体风格
王羲之的书法集前代之大成，变古朴为秀美，开创了中国书法的新纪元。其书风被形容为：

> "龙跳天门，虎卧凤阙" —— 形容其书法既有龙腾虎跃的动态美，又有稳健端庄的静态美。

> "飘若浮云，矫若惊龙" —— 形容其行书飘逸灵动，笔势矫健。

### 艺术特点
1. **中和之美**：不激不厉，刚柔并济，达到中国美学"中和"的最高境界
2. **变化无穷**：同样的字在不同位置写法各异，如《兰亭序》中21个"之"字无一相同
3. **自然天成**：笔法精妙却不见刻意，如同自然生成
4. **诸体皆精**：楷、行、草、隶、篆无所不能，尤擅行书

### 书体贡献
- **楷书**：摆脱隶书痕迹，确立楷书基本范式
- **行书**：将行书推向艺术巅峰，《兰亭序》成为千古绝唱
- **草书**：规范今草，为草书发展奠定基础

## 四、代表作品

### 天下第一行书 —— 《兰亭序》
| 项目 | 内容 |
|------|------|
| **年代** | 东晋永和九年（353年），王羲之50岁 |
| **背景** | 三月三日上巳节，与谢安、孙绰等41人在会稽山阴兰亭举行修禊之会，饮酒赋诗，乘兴而作 |
| **内容** | 记述兰亭雅集的盛况，描绘山水之美，抒发对人生无常的感慨 |
| **艺术成就** | 笔法精妙，变化无穷，21个"之"字无一雷同；结构自然，气韵生动；全篇324字，被誉为书法史上不可逾越的高峰 |
| **真迹去向** | 据传被唐太宗李世民所得，太宗极爱之，死后随葬昭陵 |
| **传世版本** | 唐代冯承素摹本（"神龙本"）最为精妙，现藏北京故宫博物院 |

### 楷书代表作
| 作品 | 特点 |
|------|------|
| 《黄庭经》 | 小楷精品，结构严谨，气韵高古 |
| 《乐毅论》 | 小楷名帖，笔力精到，为历代学书者所重 |
| 《东方朔画赞》 | 楷书佳作，体格雄健 |

### 草书代表作
| 作品 | 特点 |
|------|------|
| 《十七帖》 | 草书经典，今草之宗，是学习草书的重要范本 |
| 《丧乱帖》 | 行草书，情感真挚 |
| 《快雪时晴帖》 | 行书短札，清逸俊美 |

### 其他名作
- **《圣教序》**：唐代怀仁集王羲之字而成，是学习王羲之行书的最佳范本
- **《姨母帖》**：早期作品，尚存古朴之意
- **《寒切帖》**：行书手札，自然天成

## 五、王羲之与"二王"

### 王献之（344年—386年）
王羲之第七子，与父并称"二王"，世称"小圣"。

| 对比项 | 王羲之 | 王献之 |
|--------|--------|--------|
| **风格** | 内敛含蓄，法度严谨 | 纵逸奔放，突破常规 |
| **创新** | 集大成者 | 创造"破体"，字之大小悬殊 |
| **代表作** | 《兰亭序》《圣教序》 | 《鸭头丸帖》《中秋帖》《廿九日帖》 |
| **影响** | 正统书学之宗 | 对后世行草影响深远 |

王献之曾劝父亲改体，认为行书不应字字独立，应一笔连贯。这一创新被称为"破体"或"一笔书"。

## 六、历史影响

王羲之的书法对后世影响极为深远：

- **唐代**：唐太宗极力推崇，确立"书圣"地位，天下学书者莫不宗王
- **宋代**：苏轼、米芾等皆学王羲之
- **元代**：赵孟頫以复古为名，实则回归王羲之传统
- **明清**：帖学一派以二王为宗，影响持续千年

在中国书法史上，王羲之的地位无人能够替代。正如后世所评：

> "汉魏以来，书法继绝，惟羲之氏，总百家之功，极众体之妙，传于献之，克绍其裘。" —— 书法史论

## 七、王体学习指南

### 学习途径
由于王羲之真迹不传，学习王体主要通过以下途径：
1. **行书**：《圣教序》（集字碑刻，字多且规范，最佳入门）
2. **小楷**：《黄庭经》《乐毅论》摹本
3. **草书**：《十七帖》

### 适合人群
- 有一定书法基础的学习者
- 行书进阶学习者
- 希望追溯书法本源的学习者

### 注意事项
王体变化精微，看似简单实则极难，需要深厚的功底才能领会其妙处。初学者建议先以唐楷筑基，再循序渐进地学习王体。
