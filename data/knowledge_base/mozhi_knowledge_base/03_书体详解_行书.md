## 一、行书概述

行书是介于楷书与草书之间的一种书体，相传由东汉刘德昇所创（传说性质）。它吸收了楷书的规整和草书的流畅，既有较好的辨识度，又具有较高的书写效率，是**最实用的书体**。

### 核心特征
- **流动牵连**：笔画之间可以有牵丝映带，气脉贯通
- **楷草之间**：比楷书活泼，比草书规整
- **可识可写**：既保持较好的可读性，又提高书写速度
- **变化丰富**：兼具实用性和艺术性

### 历史地位
行书在魏晋时期成熟，东晋王羲之的《兰亭序》被誉为"天下第一行书"，标志着行书艺术的巅峰。行书被称为"走"的书体——如人行走，不急不徐，流畅自然。

## 二、行书的分类

### 1. 行楷（偏向楷书）
- 结构基本保持楷书的方正
- 部分笔画有连笔和牵丝
- 较为工整，易于辨识
- 适合日常书写和正式场合

### 2. 行草（偏向草书）
- 连笔增多，结字趋于简练
- 部分字形采用草书写法
- 书写更流畅，节奏感强
- 艺术性更高但辨识度略降

## 三、天下三大行书

### 天下第一行书 —— 王羲之《兰亭序》
| 项目 | 内容 |
|------|------|
| **作者** | 王羲之（东晋，303-361） |
| **年代** | 东晋永和九年（353年） |
| **背景** | 三月三日上巳节，王羲之与友人在会稽山阴兰亭修禊饮酒，乘兴而作 |
| **内容** | 记叙兰亭雅集的盛况，抒发对人生无常的感慨 |
| **艺术特点** | 笔法精妙，变化无穷，21个"之"字无一雷同；结构自然，气韵生动 |
| **版本** | 真迹据传随唐太宗陪葬昭陵，传世以唐代冯承素摹本（神龙本）最为精妙 |
| **地位** | 中国书法史上最具影响力的作品，行书艺术的巅峰之作 |

### 天下第二行书 —— 颜真卿《祭侄文稿》
| 项目 | 内容 |
|------|------|
| **作者** | 颜真卿（唐代，709-785） |
| **年代** | 唐乾元元年（758年） |
| **背景** | 颜真卿为祭奠在安史之乱中殉国的侄子颜季明而作，情绪激动，悲愤交加 |
| **艺术特点** | 涂抹删改之处可见当时悲恸之情；笔势纵横奔放，不计工拙而自然天成 |
| **地位** | 以真挚情感打动人心，被称为"无意于佳乃佳"的典范 |

### 天下第三行书 —— 苏轼《寒食帖》
| 项目 | 内容 |
|------|------|
| **作者** | 苏轼（宋代，1037-1101） |
| **年代** | 宋元丰五年（1082年） |
| **背景** | 苏轼被贬黄州第三年寒食节所作，抒发人生困顿之感慨 |
| **艺术特点** | 笔法沉顿，字形忽大忽小，情绪起伏跌宕；将个人情感融入书法，开"尚意"书风先河 |
| **地位** | 宋代尚意书风的代表之作 |

## 四、行书代表书家

### 王羲之（书圣）
- **朝代**: 东晋
- **成就**: 集书法之大成，被尊为"书圣"
- **行书代表作**: 《兰亭序》《圣教序》（集字）
- **风格**: 飘逸遒劲，秀美绝伦，"龙跳天门，虎卧凤阙"
- **影响**: 后世行书无不以王羲之为宗

### 王献之（小圣）
- **朝代**: 东晋（王羲之第七子）
- **成就**: 与其父并称"二王"
- **行书代表作**: 《鸭头丸帖》《廿九日帖》
- **特点**: 笔法更加纵逸，有"破体"之称

### 颜真卿
- **朝代**: 唐代
- **行书代表作**: 《祭侄文稿》《争座位帖》
- **特点**: 以篆籀笔入行，雄浑大气，别开一面

### 苏轼
- **朝代**: 宋代
- **行书代表作**: 《寒食帖》《赤壁赋》
- **特点**: 笔丰墨饱，字形扁阔，意趣盎然

### 赵孟頫
- **朝代**: 元代
- **行书代表作**: 《洛神赋》《前后赤壁赋》
- **特点**: 直追二王，圆润流畅，法度严谨

## 五、行书学习要点

### 学习前提
行书必须以楷书为基础，建议先具备一定楷书功底后再学习行书。

### 学习方法
1. **选帖**: 推荐从王羲之《圣教序》或赵孟頫《洛神赋》入手
2. **掌握连带**: 学习笔画之间的牵丝映带，做到"笔断意连"
3. **把握节奏**: 行书讲究节奏感，要有轻重快慢的变化
4. **注意收放**: 该收则收，该放则放，不能一味连绵

### 入门推荐碑帖
| 碑帖 | 作者 | 特点 | 适合阶段 |
|------|------|------|----------|
| 《圣教序》 | 王羲之（集字） | 字字独立，便于初学 | 初学 |
| 《洛神赋》 | 赵孟頫 | 流畅优美，法度清晰 | 初学 |
| 《赤壁赋》 | 苏轼 | 意趣丰富，自然天成 | 进阶 |
| 《祭侄文稿》 | 颜真卿 | 情感真挚，气势磅礴 | 进阶 |

## 六、行书与楷书的区别

| 对比项 | 楷书 | 行书 |
|--------|------|------|
| **连笔** | 字字独立 | 可有牵丝连带 |
| **速度** | 慢 | 较快 |
| **形态** | 方正严谨 | 灵活多变 |
| **重心** | 平稳 | 可有动态变化 |
| **起收笔** | 法度森严 | 简化或连带 |
| **用途** | 正式场合 | 日常书写 |
