## 一、宣纸概述

宣纸是中国传统的古典书画用纸，因原产于唐代宣州府（今安徽泾县）而得名，被誉为"纸中之王"、"千年寿纸"。宣纸以其独特的润墨性、耐久性和抗老化性，成为书法和国画最理想的书写载体。

### 核心特点
- **润墨性好**：墨色层次分明，能呈现丰富的笔墨变化
- **耐久性强**：有"纸寿千年"之誉，可保存千年不腐
- **抗老化**：不易变色、脆化
- **柔韧性好**：不易破裂，适合装裱

### 历史地位
宣纸始于唐代，盛于明清。2009年，宣纸制作技艺被联合国教科文组织列入人类非物质文化遗产代表作名录。

## 二、宣纸的分类

### 按加工工艺分类

#### 1. 生宣
| 项目 | 内容 |
|------|------|
| **工艺** | 直接由青檀树皮和沙田稻草制成，未经加工处理 |
| **特点** | 吸水性强，墨汁迅速渗透，易产生丰富的墨韵变化 |
| **效果** | 墨色层次丰富，有"墨分五色"的效果 |
| **适合** | 写意画、行草书、泼墨等需要墨韵变化的艺术创作 |
| **注意** | 对控墨能力要求高，初学者不易掌控 |

#### 2. 熟宣
| 项目 | 内容 |
|------|------|
| **工艺** | 生宣经明矾等加工处理，降低吸水性 |
| **特点** | 吸水性弱，墨汁不易渗透，墨色稳定 |
| **效果** | 墨色清晰，边缘锐利，不易洇开 |
| **适合** | 工笔画、小楷、精细书法 |
| **注意** | 久存可能"泛矾"（出现黄斑） |

#### 3. 半熟宣（煮锤宣）
| 项目 | 内容 |
|------|------|
| **工艺** | 加工程度介于生宣和熟宣之间 |
| **特点** | 吸水性适中，兼具生宣的墨韵和熟宣的可控性 |
| **适合** | **初学者最推荐**，楷书、行书练习 |

### 常见分类对比
| 对比项 | 生宣 | 半熟宣 | 熟宣 |
|--------|------|--------|------|
| **吸水性** | 强 | 中 | 弱 |
| **洇墨程度** | 大 | 适中 | 几乎不洇 |
| **墨色层次** | 丰富 | 较丰富 | 单一 |
| **控墨难度** | 高 | 中 | 低 |
| **适合书体** | 行草、大草 | 楷行皆宜 | 小楷 |
| **适合绘画** | 写意画 | 兼工带写 | 工笔画 |

## 三、宣纸的原料

宣纸主要以两种原料制成：

| 原料 | 来源 | 作用 |
|------|------|------|
| **青檀树皮** | 皖南地区特有的青檀树 | 提供纤维韧性和骨架 |
| **沙田稻草** | 泾县特有的沙田籼稻草 | 增加柔软性和吸墨性 |

优质宣纸通常以青檀皮含量分级：
- **特种净皮**：青檀皮含量最高（80%以上），质量最好
- **净皮**：青檀皮含量较高（60-70%）
- **棉料**：青檀皮含量较低（40%以下），较柔软

## 四、宣纸的规格

| 规格 | 尺寸（约） | 用途 |
|------|-----------|------|
| **三尺** | 55×100cm | 小品、练习 |
| **四尺** | 69×138cm | 最常用规格 |
| **五尺** | 84×153cm | 中堂作品 |
| **六尺** | 97×180cm | 大幅作品 |
| **八尺** | 124×248cm | 巨幅作品 |
| **丈二** | 145×368cm | 特大幅 |

常见裁切形式：
- **四尺三开**：四尺纸裁三等份
- **四尺对开**：四尺纸裁半，即条幅
- **斗方**：正方形裁切

## 五、宣纸的选择

### 初学者建议
1. **选半熟宣**：控制难度适中，不易洇墨
2. **选四尺对开或三开**：规格适中，经济实惠
3. **选机制宣练习**：练习用机制宣纸即可，不必追求手工宣纸
4. **备少量生宣**：尝试墨色变化

### 鉴别真假宣纸
| 真宣纸 | 假宣纸（书画纸） |
|--------|-----------------|
| 纤维长，撕开后有毛绒 | 纤维短，撕口整齐 |
| 润墨层次分明 | 润墨层次差 |
| 纸白而不刺眼 | 可能过白或发黄 |
| 手感绵软有韧性 | 手感偏硬或偏脆 |
| 燃烧后呈灰白色 | 燃烧后呈黑色 |

## 六、宣纸的保养

1. **防潮**：存放于干燥通风处，避免受潮发霉
2. **防晒**：避免阳光直射，防止变脆发黄
3. **防虫**：可放置樟脑丸防蛀
4. **平放**：大幅宣纸应卷起存放，不可折叠
5. **先入纸**：新宣纸可能有火气，存放一段时间后使用效果更佳（称为"退火"）

## 七、趣味知识

- **纸寿千年**：优质宣纸可保存千年以上，故宫博物院藏唐代纸质文物至今完好
- **南唐李廷珪**：五代时宣纸已极为珍贵，"一页宣纸一两金"
- **红星牌宣纸**：泾县宣纸厂出品，为国礼用纸，最为著名
- **邓小平题字**："宣纸是我国宝贵的文化艺术遗产"
