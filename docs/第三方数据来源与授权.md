# 第三方数据来源与授权

## 书法单字字库

“墨智”的集字创作模块使用 **莒光书法字库（calligraphy-community）** 作为单字候选图片来源。为保证课程演示项目克隆后可以直接使用集字功能，项目仓库随附 `data/glyph_library/calligraphy-community/` 中的字库索引和单字图片。

本项目在 README、文档和演示中注明：

> 书法单字数据由莒光书法字库（calligraphy-community）提供；集字系统仅负责检索、筛选、排版与导出。

## 上游来源

字库自带 README 说明：其最初数据来源为 @zhuojg 整理并发布的 [Chinese Calligraphy Dataset](https://github.com/zhuojg/chinese-calligraphy-dataset)。字库内容及 `index.json` 按 **Apache License 2.0** 使用。

因此，本项目随仓库保留字库原始 `LICENSE` 与 `README.md`，并在 GitHub 仓库中保留本文件作为鸣谢。后续若分发字库文件或导出的产品，应继续遵守 Apache License 2.0 及上游数据说明。
